# 📊 Superstore Sales Dashboard

## Project Overview

This project analyzes Superstore sales data using Power BI.

## Tools Used

- Power BI
- Power Query
- DAX

## KPIs

- Total Sales
- Total Profit
- Total Quantity

## Dashboard Features

- Sales by Category
- Sales by Ship Mode
- Sales by Segment
- Profit by Sub-Category
- Monthly Sales Trend
- Monthly Profit Trend
- Top States by Sales
- Interactive Slicers

- Power BI Dashboard (.pbix)

- Dashboard PDF
