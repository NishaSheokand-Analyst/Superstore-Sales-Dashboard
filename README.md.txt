# 📊 Superstore Sales Dashboard

## 📌 Project Overview

This project analyzes Superstore sales data using Power BI. The dashboard provides insights into sales performance, profitability, customer segments, shipping modes, and regional performance.

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📂 Dataset

Sample Superstore Dataset

---

## 📈 KPIs

- Total Sales
- Total Profit
- Total Quantity

---

## 📊 Dashboard Features

- Sales by Category
- Sales by Ship Mode
- Monthly Sales Trend
- Monthly Profit Trend
- Profit by Sub-Category
- Top States by Sales
- Sales by Segment
- Interactive Filters

---

## 💡 Key Insights

- Office Supplies generated the highest sales.
- Consumer segment contributed the largest share of sales.
- Standard Class was the most commonly used shipping mode.
- A few states generated a significant portion of total sales.
- Profit varies significantly across sub-categories.
